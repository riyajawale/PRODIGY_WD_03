# Task-03: Tic Tac Toe Web Application

## Description
This is a simple Tic-Tac-Toe web application built using HTML, CSS, and JavaScript.  
It allows two players to play the game turn by turn and checks for winning and draw conditions.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Interactive game board
- Player turn indicator
- Win and draw detection
- Restart game option

## How to Run
1. Download or clone the repository
2. Open `index.html` in a web browser
